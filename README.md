# Sunrise University SIS

A secure cloud-ready Student Information System (SIS)
built with Python and PostgreSQL.

## Features
- Secure login with password hashing (bcrypt)
- Real student data management with PostgreSQL
- GPA-based student filtering
- Real-time academic analytics and visualizations
- Student performance report generation
- Complete audit logging of all user actions

## Technologies Used
- Python
- PostgreSQL
- psycopg2
- bcrypt
- python-dotenv
- matplotlib
